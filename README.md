### Prerequisites

In order to run this project you need:

1. Browser.
2. VS Code editor
3. Git and GitHub

### Setup

Clone this repository to your desired folder by running this command:

```sh
  
  git clone git@github.com:BenMKT/Portfolio.git
  ```
